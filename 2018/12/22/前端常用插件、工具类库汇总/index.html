前端常用插件,工具类库

发表于 2018-12-22 | 分类于 插件

函数库

  • Lodash就是这样的一套工具库，它内部封装了诸多对字符串、数组、对象等常见数据类型的处理函数
    Lodash（推荐）：https://github.com/lodash/lodash
    中文文档：https://www.css88.com/doc/lodash/

    动画库

Animate.css

  • CSS3 动画库，也是目前最通用的动画库。
  • https://daneden.github.io/animate.css/

Anime.js

  • 一个强大的、轻量级的用来制作动画的javascript库
  • http://animejs.com/

Hover.css

  • CSS hover 悬停效果，可以应用于链接、按钮、图片等等。
  • 在线演示：http://ianlunn.github.io/Hover/
  • 立即下载：https://github.com/IanLunn/Hover/archive/master.zip
  • GitHub地址：https://github.com/IanLunn/Hover

wow.js

  • 滚动展示动画
  • WOW.js 依赖 animate.css，所以它支持 animate.css 多达 60 多种的动画效果。
  • GitHub地址：https://github.com/matthieua/WOW
  • 在线演示：http://mynameismatthieu.com/WOW/

scrollReveal.js

  • 类似 wow.js
  • https://scrollrevealjs.org/

Magic.css

  • css3 animation动画库
  • GitHub地址：https://github.com/miniMAC/magic
  • 在线演示：https://www.minimamente.com/example/magic_animations/

move.js

  • 一个小型的JavaScript库，通过JS来控制一系列的CSS动画顺序执行，使CSS3动画变得非常简单和优雅。
  • 在线演示：https://visionmedia.github.io/move.js/
  • GitHub地址：https://github.com/visionmedia/move.js

Velocity.js

  • 一个功能齐全的 JavaScript 动画套件，包括诸如淡入淡出、滚动、滚动、停止、结束、翻转等动画效果。
  • 中文文档：http://www.mrfront.com/docs/velocity.js/index.html
  • GitHub地址：https://github.com/julianshapiro/velocity

滚动库

iscroll

  • 平滑滚动插件
  • 中文文档：https://iiunknown.gitbooks.io/iscroll-5-api-cn/content/index.html
  • GitHub地址：https://github.com/cubiq/iscroll

BetterScroll

  • iscroll 的优化版，使移动端滑动体验更加流畅
  • GitHub地址：https://github.com/ustbhuangyi/better-scroll
  • 中文文档：https://ustbhuangyi.github.io/better-scroll/doc/zh-hans/#better-scroll%20%E6%98%AF%E4%BB%80%E4%B9%88

mescroll

  • 移动端上拉刷新下拉加载
  • 中文文档：http://www.mescroll.com/

animatescroll

  • jQuery滚动到指定位置
  • https://plugins.compzets.com/animatescroll/

crossfade.js

  • jQuery图片模糊插件
  • 中文文档：http://www.dowebok.com/104.html

jQuery Scrollbox

  • 图片文字滚动插件
  • 在线演示：http://www.jq22.com/jquery-info1890

liMarquee

  • jQuery无缝滚动插件
  • GitHub地址：https://github.com/omcg33/jquery.limarquee
  • 在线演示：http://www.dowebok.com/188.html

轮播图

Swiper

  • Swiper常用于移动端网站的内容触摸滑动
  • 中文文档：https://www.swiper.com.cn/

iSlider

  • 移动端滑动组件
  • 中文文档：http://eux.baidu.com/iSlider/demo/index_chinese.html

slip.js

  • 移动端跟随手指滑动组件，零依赖。
  • 中文文档：https://binnng.github.io/slip.js/
  • GitHub地址：https://github.com/binnng/slip.js

滚屏

fullPage.js

  • 全屏插件
  • http://fullpage.81hu.com/

弹出框

layer

  • 中文文档：http://layer.layui.com/

级联选择器

jQueryDistpicker

  • 中国省市区地址三级联动jQuery插件
  • http://www.jq22.com/demo/jQueryDistpicker20160621/

时间选择器

  • https://www.layui.com/laydate/

地图

  • 百度地图：lbsyun.baidu.com/
  • 高德地图：lbs.amap.com/
  • 腾讯地图：lbs.qq.com/
